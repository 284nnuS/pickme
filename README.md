# PickMe

> Online social networking application that provides chatting, making friend service by matching the right person with users based on interests, voice records, and pictures

## Requirement

-  NodeJS >= 16
-  JDK == 11
-  Yarn
-  Maven

## Install

```sh
yarn
```

## Start development

```sh
yarn dev
```

### Frontend

```sh
yarn dev:frontend
```

### Backend

```sh
yarn dev:backend
```

## Build

```sh
yarn build
```
