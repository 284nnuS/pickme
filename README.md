# PickMe

> Online chatting networking application that provides chatting and making friend service that allows users to find the right person for them

## Requirement

-  NodeJS >= 16
-  JDK >= 11
-  Yarn
-  Maven

## Install

```sh
yarn
```

## Start development

```sh
yarn dev
```

### Frontend

```sh
yarn dev:frontend
```

### Backend

```sh
yarn dev:backend
```

## Build

```sh
yarn build
```
